# WiFi Password Extractor

A Python script for educational purposes, demonstrating how to extract saved WiFi passwords on Windows using netsh and transmit them via email.

## Features

- **Password Retrieval**: Uses netsh to fetch clear-text WiFi passwords.
- **Email Transmission**: Sends extracted data to a specified email.
- **Profile Listing**: Displays available WiFi profiles.

## Requirements

- Python 3.x
- Windows OS with netsh utility
- SMTP access for email sending

## Installation

Clone the repository and install dependencies.

## Usage

Run the script to extract WiFi passwords and send them via email. Configure email credentials securely.
