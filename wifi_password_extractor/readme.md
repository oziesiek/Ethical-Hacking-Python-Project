# WiFi Password Extractor

A Python script for educational purposes, demonstrating extraction of saved WiFi passwords on Windows using netsh and email transmission.

## Features

- **Profile Retrieval**: Lists saved WiFi networks.
- **Password Extraction**: Fetches clear-text passwords.
- **Email Notification**: Sends results to a specified email.

## Requirements

- Python 3.x
- Windows OS with netsh utility
- Gmail account for SMTP sending

## Installation

Clone the repository and install dependencies.

## Usage

Configure Gmail credentials in the script. Run it to extract passwords and send via email.
