# Wi-Fi Password Extractor

A Python script for retrieving saved Wi-Fi passwords across Windows, Linux, and macOS systems. Detects the OS and uses appropriate commands to extract credentials.

## Features

- **Cross-Platform**: Supports Windows (netsh), Linux (NetworkManager), and macOS (security).
- **Interactive Selection**: Prompts for OS choice.
- **Educational Tool**: Demonstrates system command usage for Wi-Fi data.

## Requirements

- Python 3.x

## Installation

Clone the repository and install dependencies.

## Usage

Run the script and select your operating system when prompted. It will display saved Wi-Fi SSIDs and passwords.
