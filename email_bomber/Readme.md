# Email Bomber

A Python script for sending multiple emails to a target address, provided for educational purposes to demonstrate SMTP usage and potential abuses.

## Features

- **Multiple Modes**: Predefined or custom email counts (1000, 500, 250, or user-specified).
- **Server Options**: Supports Gmail, Yahoo, Outlook, or custom SMTP servers.
- **Interactive Prompts**: Guides user input for target, credentials, subject, and message.

## Requirements

- Python 3.x
- `smtplib` library (included in Python standard library)

## Installation

Clone the repository and install dependencies.

## Usage

Run the script and follow the prompts to configure the target, mode, server, and content. It will send the specified number of emails.
