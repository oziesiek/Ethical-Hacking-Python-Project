# Python Keylogger

A Python script using pynput to capture and log keystrokes for educational purposes, demonstrating input monitoring techniques and security awareness.

## Features

- **Keystroke Capture**: Records keyboard input in real-time.
- **Logging**: Saves captured data to a file or console.
- **Cross-Platform**: Works on Windows, Linux, and macOS.

## Requirements

- Python 3.x
- pynput library (`pip install pynput`)

## Installation

Clone the repository and install dependencies.

## Usage

Run the script to start logging keystrokes. Monitor output for captured input.
