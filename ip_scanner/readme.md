# ARP Scanner

A Python script using Scapy to scan local networks via ARP requests, discovering active hosts and their MAC addresses.

## Features

- **Network Discovery**: Sends ARP requests to IP ranges.
- **MAC Mapping**: Displays IP-MAC pairs for responding devices.
- **Flexible Targeting**: Accepts single IPs or CIDR ranges.

## Requirements

- Python 3.x
- Scapy library (`pip install scapy`)

## Installation

Clone the repository and install dependencies.

## Usage

Run the script with a target IP or range (e.g., 192.168.1.1/24) to scan and display active hosts.
