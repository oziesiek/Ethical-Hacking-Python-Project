# Phone Number Information Lookup

A Python script using the phonenumbers library to retrieve detailed information about a phone number, including country, carrier, time zone, and validity.

## Features

- **Geographical Data**: Extracts country and region.
- **Carrier Info**: Identifies mobile carrier.
- **Time Zone**: Lists associated time zones.
- **Validation**: Checks number validity and provides E.164 formatting.

## Requirements

- Python 3.x
- phonenumbers library (`pip install phonenumbers`)

## Installation

Clone the repository and install dependencies.

## Usage

Modify the script with a phone number in E.164 format and run it to display the extracted information.
