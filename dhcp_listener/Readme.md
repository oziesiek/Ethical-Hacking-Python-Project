# DHCP Listener

A Python script using Scapy to capture and analyze DHCP packets on a Wi-Fi network, displaying device details such as MAC address, hostname, vendor ID, and requested IP.

## Features

- **Packet Sniffing**: Monitors DHCP requests in real-time.
- **Device Info Extraction**: Parses MAC, hostname, vendor ID, and IP requests.
- **Formatted Output**: Logs discoveries with timestamps.

## Requirements

- Python 3.x
- Scapy library (`pip install scapy`)

## Installation

Clone the repository and install dependencies.

## Usage

Run the script to start sniffing DHCP packets and display device information.
