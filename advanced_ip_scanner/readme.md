# Network Host and Service Discovery

A Python script using Scapy for discovering active hosts and open ports on a network. Performs ICMP-based host discovery followed by port scanning on identified hosts.

## Features

- **Host Discovery**: Sends ICMP Echo requests to find active devices.
- **Port Scanning**: Identifies open ports on discovered hosts.
- **Flexible Targeting**: Accepts IP addresses, ranges, or hostnames.

## Requirements

- Python 3.x
- Scapy library (`pip install scapy`)

## Installation

Clone the repository and install dependencies.

## Usage

Run the script with a target IP, range, or hostname. It will display discovered hosts and their open ports.
