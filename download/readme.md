# Malicious Script - Data Exfiltration

A Python script demonstrating data exfiltration techniques for educational purposes, simulating malware download, execution, and email transmission of captured data.

## Features

- **Malware Download**: Retrieves an executable from a remote server.
- **Execution and Capture**: Runs the malware and collects output.
- **Data Transmission**: Sends captured information via email.

## Requirements

- Python 3.x
- Access to SMTP server for email sending

## Installation

Clone the repository and install dependencies.

## Usage

Run the script to simulate the exfiltration process. Configure URLs and email details for testing.
