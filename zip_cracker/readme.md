# ZIP File Password Cracker

A Python script for cracking password-protected ZIP files using a dictionary attack, iterating through a wordlist to find the correct password.

## Features

- **Dictionary Attack**: Tries passwords from a provided wordlist.
- **Built-in Library**: Uses Python's zipfile module.
- **Success Notification**: Displays the found password and extraction details.

## Requirements

- Python 3.x
- ZIP file and password wordlist in the same directory

## Installation

Clone the repository and install dependencies.

## Usage

Run the script to attempt cracking the ZIP file with the wordlist. It will output the password if found.
