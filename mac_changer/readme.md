# MAC Address Changer

A Python script for educational purposes, demonstrating how to change the MAC address of a network interface using command-line utilities on Unix-like systems.

## Features

- **Interface Selection**: Specify target network interface.
- **MAC Modification**: Update to a new MAC address.
- **Verification**: Confirms successful change.

## Requirements

- Python 3.x
- Unix-like system (Linux or macOS) with `ifconfig` utility

## Installation

Clone the repository and install dependencies.

## Usage

Run the script with options for interface (`-i`) and new MAC (`-m`). It will change and verify the MAC address.
